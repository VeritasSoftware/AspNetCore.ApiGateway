# API Gateway as a RESTful Microservice Facade

## Client for Api Gateway built using AspNetCore.ApiGateway package.

API Gateway Client library helps you to easily interact with the Api Gateway built using [**AspNetCore.ApiGateway**](https://www.nuget.org/packages/AspNetCore.ApiGateway) package.

It provides a simple and intuitive interface for making requests to the API Gateway and handling responses.
